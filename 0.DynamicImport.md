​Tips Tambahan untuk Vite
​Karena kamu pakai Vite, kamu bisa memantau ukuran file-mu dengan plugin visualizer. Tambahkan ini di devDependencies jika ingin melihat file mana yang paling "makan tempat":
npm install rollup-plugin-visualizer --save-dev
​Setelah itu, tambahkan di vite.config.ts. Ini akan sangat membantu kamu memutuskan library mana lagi yang perlu di-dynamic import.
